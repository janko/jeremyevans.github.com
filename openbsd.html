OpenBSD

I'm an OpenBSD developer (jeremy@), responsible for OpenBSD's ruby ports.

In addition to ruby related ports, I'm also the maintainer of the following OpenBSD ports:

  • audio/aqualung: advanced music player
  • audio/mp3applygain: apply gain information to mp3 audio data
  • comms/c3270: curses-based IBM 3270 terminal emulator
  • databases/postgresql: relational database
  • databases/postgresql-pllua: Lua procedural language support for PostgreSQL
  • databases/postgresql-previous: relational database, previous version to support pg_upgrade
  • textproc/hyperestraier: high performance full-text search system
