Ruby

I'm a Ruby committer, mostly focusing on general bug fixing.

I maintain many Ruby libraries, some of the more popular ones are:

  • Sequel: The Database Toolkit for Ruby
  • Roda: Routing Tree Web Toolkit
  • Rodauth: Ruby's Most Advanced Authentication Framework
  • Erubi: ERB Template Engine (used by most Ruby web frameworks)
  • american_date: American style month/day/year date parsing for Ruby
  • minitest-hooks: before(:all)/after(:all)/around/around(:all) hooks for minitest
  • ruby-warning: Custom processing for Ruby warnings
