Notes:

Most ragel machines are built out of other ragel machines, such as this one named clf_date_time, which is a machine for the entire Apache Common Log format.

This machine accepts the clf_date format optionally followed by a colon and the clf_time format and whitespace.

If the input string ends in one of the clf_date_time's final states, the set_parser_clf action will be called.
